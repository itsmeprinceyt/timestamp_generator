# Timestamp Generator

A minimal, live-updating timestamp generator with a retro terminal UI. Click to copy.

## Format

```
YYYYMMDDHHMMSSmmm
```

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- react-hot-toast

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.
